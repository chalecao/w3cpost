# 题目

给定一个 n 位数的整数，翻转顺序，返回翻转后的整数

# 示例

比如 456 翻转后 654， 100 翻转后 1

# 分析

问题是，001和01 要变成 1，最后转换下就好了

# 参考答案

### 思路1

字符串翻转，最后转成数字就好了

<pre class="EnlighterJSRAW" data-enlighter-language="null">function reverse(num){
    return +((""+num).split("").reverse().join(""))
}</pre>
